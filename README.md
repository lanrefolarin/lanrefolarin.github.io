# lanrefolarin.github.io

Personal portfolio site for Lanre Folarin, Senior Business Systems Analyst at United BioSource (UBC), building toward Product Management.

Live site: [lanrefolarin.github.io](https://lanrefolarin.github.io/)

## About

A single-page portfolio covering work experience, case studies, certifications, skills, and writing. Built as a static site hosted on GitHub Pages.

The design treats the page as a traceable requirements document. Sections carry requirement IDs (REQ-00 through REQ-06), styled as a light blueprint: ink on paper, with an ultramarine analysis accent and an amber decision accent.

## Features

- Word-by-word hero name reveal on load, with a scroll progress bar and a scrolling marquee of industries and tools
- Sticky-stacking project case studies: each card pins in place and scales back as the next one slides over it
- Animated KPI counters and an interactive tenure chart; clicking a bar scrolls to the matching experience entry
- Case studies framed as User Need, Decision, and Impact
- Credential cards link directly to Accredible verification pages (IIBA and Product School)
- Magnetic hover effect on primary buttons
- Responsive down to mobile, keyboard focus styles, and `prefers-reduced-motion` support

## Tech

- HTML5, CSS3, vanilla JavaScript (no frameworks, no libraries)
- Single `index.html` file, no build step
- Hosted on GitHub Pages
- Fonts via Google Fonts (Bricolage Grotesque, Inter, IBM Plex Mono)
